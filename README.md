# Proyecto Web SENATI - Plataforma de Admisión

Este proyecto es una Landing Page moderna y de alto rendimiento diseñada para **SENATI**, enfocada en la captación de nuevos estudiantes y la presentación de la oferta académica 2025-II.

---

##  Tecnologías Utilizadas

El desarrollo se centró en la creación de una interfaz limpia, oscura y tecnológica utilizando:

* **HTML5:** Estructura semántica para SEO y accesibilidad.
* **CSS3 (Custom Properties):** Sistema de diseño basado en variables para una paleta de colores coherente (Verdes industriales y Dorados).
* **JavaScript (Vanilla):** Implementación de *Intersection Observer* para animaciones de revelado al hacer scroll.
* **SVG Dinámico:** Uso de texturas de ruido (noise texture) para profundidad visual.

---

## Organización del Proyecto (Arquitectura)

Para mantener el código profesional y escalable, el proyecto se organiza de la siguiente manera:

* **Diseño Visual:** Implementación de un sistema de rejilla (Grid) y Flexbox para asegurar que la página sea responsiva.
* **Secciones Clave:**
    * *Hero Section:* Impacto visual con tipografía display.
    * *Métricas de Éxito:* Visualización de datos de empleabilidad mediante barras animadas.
    * *Proceso de Admisión:* Guía paso a paso mediante pseudo-elementos CSS.
* **Optimización:** Uso de fuentes de Google Fonts (`Bebas Neue` y `DM Sans`) cargadas de manera eficiente.
